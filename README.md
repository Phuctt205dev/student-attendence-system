# Đồ án môn học NT208.Q21.ANTT - Lập trình ứng dụng Web nhóm 6

## Đề tài

Website điểm danh sinh viên, ra đề kiểm tra và chấm bài tự động

- Xây dựng hệ thống quản lý lớp học đa nền tảng với chức năng điểm danh bằng mã QR/nhận diện khuôn mặt và thi trực tuyến, sử dụng React, Firebase Auth/Firestore và phân quyền truy cập theo vai trò cho Admin, Giảng viên và Sinh viên.
- Phát triển ngân hàng câu hỏi, phân công bài kiểm tra, chấm điểm tự động trên web/mobile, xuất dữ liệu ra Excel, và hỗ trợ tạo/trích xuất câu hỏi từ tài liệu học tập. Chấm bằng tự động bằng OMR.
